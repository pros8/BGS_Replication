# Репликация статьи "The Behavioralist Goes to School" (Levitt et al., 2016)

Данный репозиторий содержит код и материалы для частичной репликации эмпирического исследования влияния краткосрочных стимулов на академическую успеваемость школьников на основе работы *Levitt, S. D., List, J. A., Neckermann, S., & Sadoff, S. (2016). "The Behavioralist Goes to School: Leveraging Behavioral Economics to Improve Educational Performance"*.

**Авторы репликации:** Григорьев Матвей, Кравец Иван, Просин Олег

## Описание проекта

Цель проекта — воспроизвести основные результаты оригинального исследования, переведя алгоритм анализа из среды Stata в R. В рамках работы мы проверяем, насколько устойчивы эффекты немедленных и отложенных вознаграждений, а также анализируем гетерогенность воздействия в зависимости от возраста учащихся, типа сдаваемого предмета и фрейминга наград (выигрыш или потенциальная потеря).

### Успешность репликации
Репликация прошла успешно. Нам удалось со 100% точностью воспроизвести:
* описательные статистики и баланс выборки (Таблицы 3, 4, 5). В ходе работы была обнаружена и скорректирована техническая ошибка в оригинальном do-файле авторов для округа Chicago Heights (отсутствие группы CSM при формировании контрольной выборки);
* эффекты немедленных вознаграждений во всех 12 спецификациях (Таблица 6);
* эффекты отложенных вознаграждений (Таблица 7);
* гетерогенные эффекты по возрасту (Таблица 8), предмету (Таблица 9) и полу (Таблица 10);
* эффекты спилловера на будущую успеваемость (Таблица 11);
* также составлена подробная рецензия на оригинальную статью (файл находится в директории `docs/`).

Все выявленные в процессе работы расхождения с оригинальным текстом статьи (например, разница в 1 наблюдение в моделях спилловера из-за разной обработки `NA` при сравнении дат в R и Stata) задокументированы и объяснены в итоговом отчете.

## Структура репозитория

* `data/` — директория с исходными данными (`all_BGS_data.dta`).
* `docs/` — директория с сопроводительной документацией:
  * `report_of_replication` - итоговый отчет по репликации (PDF).
  * `article_review.pdf` - рецензия на оригинальную статью.
  * `article_origin.pdf` - текст оригинальной статьи авторов.
* `replication_report.Rmd` — основной файл с кодом пайплайна.
* `replication_report.html` — сгенерированный HTML-отчет.
* `BGS_Replication.Rproj` — файл проекта RStudio.

## Инструкция по запуску (Воспроизводимость)

Для того чтобы полностью воспроизвести результаты анализа, выполните следующие шаги:

1.  Клонируйте репозиторий на свой локальный компьютер:
    ```bash
    git clone [https://github.com/pros8/BGS_Replication](https://github.com/pros8/BGS_Replication)
    ```
2.  Откройте файл проекта `BGS_Replication.Rproj` в среде RStudio.
3.  Убедитесь, что файл данных `all_BGS_data.dta` находится в папке `data/`.
4.  Откройте файл `replication_report.Rmd`.
5.  Установите необходимые пакеты (если они не установлены):
    ```R
    install.packages(c("tidyverse", "haven", "fixest", "modelsummary", "knitr", "lubridate"))
    ```
6.  Нажмите кнопку **Knit** (или используйте сочетание клавиш `Ctrl+Shift+K` / `Cmd+Shift+K`) для компиляции кода и генерации итогового отчета.

## Используемые библиотеки R
* `tidyverse` — предобработка и манипуляция с данными.
* `haven` — импорт данных формата `.dta` (Stata).
* `fixest` — оценка регрессионных моделей с фиксированными эффектами и кластеризованными стандартными ошибками.
* `modelsummary` и `knitr` — генерация академических таблиц.
* `lubridate` — обработка форматов дат для оценки эффектов спилловера.

## Декларация об использовании ИИ
В процессе перевода аналитического пайплайна из Stata в R, а также для отладки эконометрических функций (в частности, спецификаций пакета `fixest` и форматирования таблиц `modelsummary`) использовалась языковая модель Gemini 3.1 Pro. Инструмент применялся в качестве ассистента для написания и оптимизации кода.
